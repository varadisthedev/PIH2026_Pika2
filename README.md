# RentiGO: Hyper-Local Neighborhood Explorer

**Redefining Neighborhood Sharing.**  
RentiGO is a premium, seamless peer-to-peer rental marketplace designed to unlock the value of underused gear within your community. Whether it's high-end creative equipment or everyday essentials, RentiGO bridges the gap between ownership and access with intelligence-driven insights and a secure, aesthetic interface.

##  Key Features
- **Hyper-Local Discovery**: Advanced geospatial filtering to find gear exactly where you are.
- **Trusted Secure Payments**: Frictionless transactions integrated with Razorpay.
- **Identity-First Trust**: Secure user management and authentication powered by Clerk.
- **Real-Time Synergy**: Instant neighborhood communication via integrated messaging.

## 🛠️ Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS 4, Lucide React
- **Backend**: Node.js, Express, MongoDB
- **Auth**: Clerk (Auth & Security)
- **Payment Infrastructure**: Razorpay API
- **Messaging**: Socket.io
- **Deployment**: Vercel & Render
- **Geospatial**: Leaflet & OpenStreetMap
